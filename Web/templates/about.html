About FGO Automation

T

he FGO Automation is a project offers you some functionalities to handle with exhausting works while playing Fate Grand Order. Originally, the project is developed based on IOS devices. However, the IOS is not as open as android system, there is no solution to control IOS devices but a bluetooth mouse. In early 2022, FGO simplified Chinese team released cross platform login functionality, it is possible to do the exhausting IOS things with Android emulator. The system functionalities only contain ones which I need, thus, it is impossible to satisfy all of the user. But if you are skilled at programming, you can develop other functionalities based on this project. Computer vision technology, multi threading, web server and html knowledge is required for this program. The source code can be found on Github.
