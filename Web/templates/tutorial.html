Quick start

In the index page, you can easily choose the kind of task, like battle or friend point summon. Just press the create task button to create a task and then use start, pause, resume buttons to control task status. But do remember to correctly connect your android device and configure the system and battle script first before starting a task.

About docker and ToolBox

The docker contains multiple icons link to all of the system pages and some of the useful and frequently visit website. It is also developed with amazing animation. You can easily jump to the webpage if necessary.

The ToolBox is another useful function which let you jump to any system page through a Mac OSX like folder. just click the blanket to spread or collapse the inner icons. Trust me, you will be addictive to it.

Task event operation

The task system on the index page presently supports two kinds of functions, the auto battle functionality according to self-defined battle script file, and friend point summon functionality.

A task type should be firstly chosen in the selectBox, then register this task to the system by click create task button. A message box will be shown on the webpage to present the result. Then use start button to start task. lt is also possible to pause the task and resume it when in processing. When another task need to be created, do remember to stop the previous one, otherwise the create process cannot success.

The auto battle functionality needs to configure the battle script before using. For details, see the Battle script operation part.

Modify system configuration

the setting page manages all of the issues related to system configuration. You can set email notice information, battle preferences and edit battle script in this page.

Battle script operation

Just use the well-designed battle script generator to create, edit, overwrite the script file. The battle script file is a JSON file which can be automatically phrased by the FGO battle engine. You cann also write the script file by yourself if you are skilled at JSON format, just put your file under FGO_func/Config/battle folder.
